Дистрибутив системы ЭлФин состоит из 2 файлов:
	- Файл efsrv.rar содержит прототип пустой базы данных системы - каталог Server с подкаталогами
	- Файл stations.rar содержит прототипы рабочих станций
Ставить систему можно куда угодно, в том числе на системный диск (C:), хотя это - плохая практика. Для определенности в данном документе предполагается, что вы устанавливаете ее на свой диск d:.
Система поставлется настроенной на 1 несетевой магазин с 2 кассами.
Система предполагает наличие установлнного MS Office для печати большинства форм.
Система ориентирована на использование клиент-серверной СУБД ADS (Advantage Database Server, Sybase) и включает в себя локальный 
бесплатный эмулятор ADS (до 5 рабочих станций). Чтобы попробовать, или на небольшой магазин -- этого хватает, однако для нормальной 
устойчивой работы системы в нагруженном магазине горячо рекомендуется раздобыть (скачать с torent'ов, или вовсе купить) и установить 
"настоящий" ADS с версией не ниже 10.

Для ручной установки системы из дистрибутива надо:

1. Создать на любом диске (например, d:) папку(каталог), в которой Вы хотите видеть базу данных (например, d:\#efsrv)
2. Назначить этой папке логический диск S: (локально - командой "subst s: d:\#efsrv", в сети - объявив эту папку разделяемым ресурсом с именем S:)
3. Разархивировать файлы из efsrv.rar в эту папку, сохраняя структуру подкаталогов. В результате должен появиться S:\Server - серверный каталог системы
4. Скопировать из архива stations.rar на рабочие компьютеры (например, в диск d:) следующие каталоги прототипов рабочих станций:
	\station.hbr (консольная рабочая станция), копируется на каждый рабочий компьютер оператора
	\station.elf (рабочая станция Win32), копируется на каждый рабочий компьютер оператора
	\station.e00 (демон автообновления, репликаций, обмена с ЕГАИС и мого чего еще), копируется только на одну станцию
	\station.fr (консольный кассовый модуль), копируется на каждый кассовый аппарат
	Файлы, содержащиеся в папке SYSTEM32 из архива, необходимо переписать в системный каталог MS Windows (c:\windows\system32 для WinXP, c:\windows\SysWOW64 для 64-битных Win7-10) каждого из рабочих компьютеров, или в каждый из вышеперечисленных каталогов.
5. Для каждого из скопированных в п.4 подкаталогов создать иконку на рабочем столе.
   Исполняемые файлы ("терминалы"):
      - Для d:\station.hbr, d:\station.fr - "Valhda.exe". Это консольный терминал, поэтому в Свойстве иконки следует настроить 
        Расположение -> Размер буфера экрана, Размер окна и Шрифт. Шрифт лучше выбрать точечный, минимальный размер буфера экрана 80 * 25.
      - Для d:\station.elf - "ElFin.exe /ADS=YES"
      - Для d:\station.e00 - "ElFin.exe /AUTO=XXXExpImpTXT/ADS=YES"
8. На кассовом компьютере установите драйвер соответствующего фискального регистратора (ФР), если его там нет. Поддерживаются
   драйверы СПАРК (UDSpark), АТОЛ (до 10 включительно), Штрих (с эмуляцией через АТОЛ). Система также работает с СПАРК-130Ф.
9. Войдите в систему с одной из рабочих станций и настройте подключения имеющегося у Вас оборудования (см. config.txt).
	- Административный вход в систему на любой рабочей станции, включая кассовый аппарат (код/пароль): 101/0
	- Вход в кассовый модуль на кассовом компьютере (код/пароль): 911 или 912/0 (это код/пароль самого кассового аппарата,
		для пробития чеков также понадобится код/пароль кассира 114/0).
